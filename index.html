Install & run

One file, no required dependencies. Or run it as an MCP server for any MCP host (Claude Code, Cursor, Windsurf, Codex, Gemini).

pip install agora-mnemo        # the library + the MCP server
mnemo-mcp                      # start the stdio MCP server

Correction is a first-class operation — measured across systems

Any memory layer can store and retrieve. The harder, less-benchmarked property is integrity: when a fact is corrected, can the store undo the correction on command? Measured against mem0 and Graphiti in their native configs, with a shared ground-truth-blind judge.

value-obscuring revert · undo a correction from an unmarked "go back" (n=20)success95% CI
mnemo (route / revert)0.75[0.53, 0.89]
mem0 2.0.11 (native, gpt-4o-mini)0.20[0.08, 0.42]

Only mnemo exposes a channel to undo a correction on command; the CIs against mem0 do not overlap. We lead with the cell we don't win: echo-resurrection is a tie — all three defend a restated stale value. A narrow, adversarial, command-driven cut, not a general "mnemo is better" claim. What it does

Value-ranked recall

Retrieval by relevance × accrued value, per-type decay, and a lexical+semantic auto-mode (BM25 + embeddings fused by RRF). Zero-dep by default.

Correction & erasure channel

revert(), retract_lineage(), echo_guard, and forget_subject() — undo, cascade a lineage-aware retraction, and prove deletion with a signed tombstone.

Tamper-evident receipts

verify_writes() over a hash-chained, optionally Ed25519-signed write log — prove the memory wasn't edited out of band.

Poison-resistant influence

recall(influence_only=True) restricts what may drive an action to corroborated memories — drops a single-instance retrieval-poisoning hijack to ~0%.

Consolidation

Capacity-aware clustering, contradiction flagging, and a self-maintaining second brain (dead-link / orphan / stale repair).

Framework adapters

Drop-in backends for OpenAI Agents, AutoGen, LangGraph, LlamaIndex, Google ADK, and Pydantic AI — each carrying the current-truth recall differentiator.

The MCP server

A Model Context Protocol server over stdio, published to the official MCP registry as io.github.DanceNitra/mnemo. Twelve tools:

rememberrecallroute revertforgetconsolidate check_conflictcontradictionscredit value_by_cohortconsolidate_clusterssleep

Config: MNEMO_PATH (the memory file), MNEMO_ECHO_GUARD (block a restated retired value from resurrecting a corrected fact).

Measured, not assumed — every design rule ships with a runnable probe.
